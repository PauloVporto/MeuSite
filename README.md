# Portfólio Paulo Vicente Porto

Site estático criado para hospedagem na Vercel.

## Arquivos

- `index.html`: estrutura da página
- `styles.css`: identidade visual e responsividade
- `script.js`: carregamento automático dos repositórios públicos do GitHub

## Publicar na Vercel

1. Envie estes arquivos para um repositório no GitHub.
2. Acesse a Vercel e importe o repositório.
3. Escolha a opção de projeto estático.
4. Publique sem necessidade de build customizada.
